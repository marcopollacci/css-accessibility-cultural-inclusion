margin[block|inline]

Let's Code 👨‍💻

  
    .example {
      margin-block-start: 10rem; //equivalent to margin-top
      margin-block-end: 10rem; //equivalent to margin-bottom
      
      margin-inline-start: 10rem; //equivalent to margin-left
      margin-inline-end: 10rem; //equivalent to margin-right
    }
  
    .example-better {
      margin-block: 10rem; //equivalent to margin-top and margin-bottom
      margin-inline: 10rem; //equivalent to margin-left and margin-right
    }

  • 1. Noto Fonts (Google Noto) Descrizione: Noto è una famiglia di font sviluppata da Google per coprire tutte le lingue del mondo. Il nome "Noto" deriva da "No Tofu", riferendosi all'obiettivo di eliminare i caratteri quadrati (tofu) che appaiono quando un font non supporta un determinato carattere. Esempi: Noto Sans, Noto Serif, Noto Mono.
  • 2. Arial Unicode MS Descrizione: Arial Unicode MS è una versione estesa del popolare font Arial, progettata per includere un'ampia gamma di caratteri Unicode. Utilizzo comune: Documenti multilingue, specialmente in ambienti aziendali.
  • 3. DejaVu Fonts Descrizione: DejaVu è una famiglia di font open-source che estende il font Bitstream Vera, aggiungendo supporto per un'ampia gamma di caratteri Unicode. Esempi: DejaVu Sans, DejaVu Serif, DejaVu Sans Mono.
  • 4. Code2000 Descrizione: Code2000 è un font shareware progettato per supportare il maggior numero possibile di caratteri Unicode. Utilizzo comune: Ricerca linguistica, documentazione tecnica.

E se non posso usare un font multilingual?
